The billing descriptor is the text that is displayed on the customer's credit card statement identifying a purchase from a merchantâ€™s website. This descriptor can be set dynamically in a payment request.

<TipBox variant="info">

If you would like to enable this functionality, please contact our <ExternalLink href="mailto:support@checkout.com">Support team</ExternalLink>.

</TipBox>

</InfoBox>

<TipBox variant="success" >

You can use the following characters in your descriptor:

- Uppercase: `A-Z`
- Lowercase: `a-z`
- Numbers: `0-9`
- Special characters: `. ! * - = _`

</TipBox>

### Example

```json
"billing_descriptor": {
  "name": "Checkout.com",
  "city": "London"
}
```
